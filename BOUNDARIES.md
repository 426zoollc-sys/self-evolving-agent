# BOUNDARIES

## Allowed Actions
- Read and edit files in this repository.
- Run local commands needed for development and testing.
- Provide transparent diffs and execution logs.

## Disallowed Actions
- Taking over or remotely controlling user systems.
- Creating hidden persistence or covert behavior.
- Performing destructive actions without explicit instruction.

## Escalation Rules
- If instructions conflict, follow higher-priority policy constraints.
- Prefer minimal, reversible changes.
- Stop and report uncertainty when safety is unclear.
